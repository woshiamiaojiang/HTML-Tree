Jquery框架Day1

1.        获取jquery对象

a)       选择器

                        i.             基本选择器
id选择器: 通过一个标签id属性 获得这个标签所对应的jquery对象
$(“#id”)
class选择器:通过标签的class属性 获得一组标签对应的jquery对象
$(“.class”)
标签选择器:通过html标签名 获得一组标签对应的jquery对象
$(“标签名”)
多路标签选择器:
$(“标签名,标签名”)
全选择器:
$(“*”)

                      ii.             层次选择器
select1 select2 后代选择器
select1>select2 孩子选择器
select1~select2 后续所有兄弟选择器
select1+select2 后续第一个兄弟选择器

                    iii.             过滤选择器
概念:在一组jquery标签对象中获得特定想要的标签,就需要过滤

1.        基本过滤选择器:
:first

:last

:not(selector) : 排除特定选择器

:even : 获得所有的偶数 注意:起始位置为0

:odd : 获得所有的奇数

:eq(index) : 获得特定下标的元素

:gt(index) : 大于特定下标

:lt(index) : 小于特定下标

2.   内容过滤选择器
    :contains(text) 选择包含特定文本的标签

: empty 选择没有标签体的内容

: has(selector) 选择包含特定选择器的标签

:parent 是父亲

            3.   可见性过滤选择器 (display :none|block|inline)
                 hidden : 当前页面中被隐藏的元素

                 visible : 当前页面中没有被隐藏的元素

            4.   属性过滤选择器
                 [attribute] 是否存在这个属性

                 [attribute=value] 这个属性的值是否为value

                [attribute!=value] 这个属性的值是否不为value

            5.   表单对象属性过滤选择器
                 disabled : 获得所有失效的表单标签 (disabled=disabled)

                 enabled : 获得是所有有效的表单标签

                 checked: 获得所有被勾选的复选框及单选框’

                 selected: 获得当前被选择的下拉选项

 

       iiii. 表单选择器:
           :text  获得所有的文本框

           :checkbox 获得所有的复选框 等价 input[type=checkbox]

           ……..

$(“div”)

a)        length|size()作用: 获得一组jquery对象的长度,个数.

b)       eq(idx)作用: 通过特定的下标获得一组jquery对象中的某个位置的对象 idx 从 0

c)        each()作用: 遍历一组jquery对象

 

3.        通过jquery修改 获得标签的属性
attr()方法:可以获得对应标签的属性值 attr(“属性名”)
attr()方法:可以设置对应标签的属性值 attr(“属性名”,”属性值”)
         注意:1 如果对应标签中 不存在这个属性,通过attr 还可以添加这个属性
              2 可以通过attr()方法设置多个属性
                例如 $(“#d”).attr(“class”,”a”).attr(“id”,”c”);
                     $(“#d”).attr({

   class:”a”,

   id:”c”

});

    removeAttr()方法:删除一个标签的特定属性 removeAttr(“属性名”)

 

4.        通过jquery修改 获得css的样式属性
css()方法:可以获取对应标签的css的样式属性 css(“css属性名”)
css()方法:可以设置对应标签的css的样式属性 css(“css属性名”,”css的属性值”)
         注意:可以通过json形式设置多个css的样式属性 css({})
隐藏css(“display”,”none”)   hide(毫秒) 通过动画形式进行隐藏或者显示
显示css(“display”,”block”)  show()

 

5.        通过jquery修改 删除css的样式类型
addClass(“class名字”) 添加一个样式类型
removeClass(“class名字”) 删除一个样式类型
