页面指令

<%@ page  属性="值" 属性2="值2" %>

用于定义对整个JSP页面都有效性的特性说明。

       1) pageEncoding="UTF-8"   页面编码

       2)contentType        设置jsp页面的内容类型和字符集

              contentType="text/html;charset=UTF-8"

       3)import                当前JSP页面中要导入的类或包

              import="java.util.*"

              import="java.uitl.Date"

              import="java.util.*" import="java.io.*"

              import="java.util.*,java.io.*"

       4)isErrorPage="true|false(默认)"  当前JSP能否进行异常处理提示

示例：

<%@ page pageEncoding="UTF-8" contentType="text/html"%>

<%@ page isErrorPage="true" %>

出错啦！！！

       5)errorPage="url地址"  如果JSP运行中出现异常，则转到url地址指定的错误页面执行

示例：

<%@ page errorPage="/error.jsp" %>

<%int result = 5/0; %>

运行结果：

       6)session="true(默认)|false"  当前JSP中是否能直接使用session对象

              servlet中使用session: getSession(true)              懒汉式获取session

              jsp中使用session:直接使用                      饿汉式

在JSP中不使用session的默认值，获取 session的方法：

<%@ page session="false"%>

<%

               // jsp中不自动获取session对象

               HttpSession session = request.getSession(true);

           %>

<%=session.getId() %><br/>

       7)language="java"         当前JSP中的动态编程语言

       8)isELIgnored="true|false"           当前JSP中是否忽略EL表达式语言

              在tomcat6.X中，默认是false。

包含指令

<%@ include file="url路径" %>

       在当前JSP页面中，包含另外一个页面。

       特点：静态包含/源代码包含/编译前包含

       注意：包含和被包含的网页中，不允许定义同名的局部变量。

logo.jsp

<%@ page pageEncoding="UTF-8"contentType="text/html"%>

<%

    int num = 33;

    %>

<html>

        <head>

           <title>公共页面</title>

        </head>

        <body>

           <img src="/jsp_day1/image/logo.gif"/>

           <hr/>

        </body>

</html>

动作元素

知识点

include动作

作用：在当前JSP页面中，包含另外一个页面。

等价于servlet中的：

       reqeust.getReuqestDispatcher("/url").include(request, response);

语法：

<jsp:include page="url路径" />

<jsp:include page="url路径" ></jsp:include>

特点：动态包含/结果包含/编译后包含

示例：

<%

        int num = 5;   //使用include动作包含时，正确

%>

<jsp:include page="/logo.jsp"/>

作用：在当前JSP页面中，包含另外一个页面，同时传递请求参数。

语法：    <jsp:include page="">

                     <jsp:param name="" value=""/>

              </jsp:include>

// 在被包含页面中，获取请求参数

       str = request.getParameter("");

示例：

jsp1.jsp

<jsp:include page="/logo.jsp">

        <jsp:param name="name" value="tom"/>

</jsp:include>

forward动作

作用：在当前JSP中，跳转到另外一个页面。

等价于servlet中的：

       reqeust.getReuqestDispatcher("/url").forward(request, response);

语法：

       <jsp:forward page="url路径"/>

       <jsp:forward page="url路径"></jsp:forward>

 

作用：在当前JSP页面中，跳转到另外一个页面，同时传递请求参数。

语法：

       <jsp:forward page="url路径">

              <jsp:param name="" value=""/>

       </jsp:forward>

// 在转向的页面中，获取请求参数

       str = request.getParameter("");
