JavaScript 模块四 BOM编程

    与DOM类似，BOM（browser object model）全称为“浏览器对象模型”。是一组和浏览器相关的内置对象，以及对象的方法。在这些对象和方法中，window是最上层对象。一个window对象代表一个浏览器窗口。之前的document是window的属性，在使用window对象的方法或者属性时，window可以省略不写。

       BOM提供的常见功能函数和属性：

    1）var w = window.open("url"); 

          打开一个新的浏览器窗口，并在新窗口里显示url指定资源。w代表新窗口

    2）window.close();  ---> 关闭浏览器窗口

       3）var id = window.setTimeOut(fun,time);

          设定time毫秒后，执行fun函数。id是定时器的编号。

          window.clearTimeOut(id);

       4）var id = window.setInterval(fun,time);

           周期性执行fun函数，每次间隔time毫秒。

              window.clearInterval(id);

    5）提示框

           window.alert("");   ----> 警告框

              var ba = window.confirm("");  ---> 确认框

              var str = window.prompt("提示信息"); ---》提示输入框

    6）location属性

       从js代码跳转到servlet程序，或者其他jsp页面，html页面借助代码:  

       --->  window.location.href="下一个程序的uri";
