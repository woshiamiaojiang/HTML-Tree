第7讲 JavaScript中的对象类型

    JavaScript中也可以使用对象。但是对对象的使用不像Java中那样严谨，没有严格的封装，多态，继承的特性。JavaScript里没有类，我们可以将对象看成是包含多种数据的集合。

1. 对象创建

    通常我们会采用JSON串的方式创建对象，对象创建时没有固定的模板，属性可以随时任意添加。具体语法如下：

    var 对象名 = { 属性名:属性值 ，属性名：属性值, .......}；

    还有另外一种创建方式，类似Java一样new Object()创建。

    var student = new Object();

   student.id = 1 ;

   student.name = “liucy”;

    上例中我们就创建了一个student对象。由id 和 name 两个属性组成。

2. 对象访问

  1）通过 对象名.属性名 访问；---->  alert(stu.age);

  2）通过 对象名["属性名"]访问；---->  alert(stu["sex"]);

  3）遍历对象的所有属性，使用for..in语法

             for( 变量名 in 对象){

                   alert( 对象名[变量名] );

                }

     循环每执行一次，都会从对象中取得一个属性名赋值给变量。

3. 对象的方法

    JavaScript里对象只有属性，没有功能。但是我们可以通过代码模拟java中的对象，使其含有属性和方法（函数）。

      var teacher ={ name:"luxw" , age:18 };

          teacher.getName = function(){  return this.name; }

          teacher.setName = function(name){ this.name=name; }

此处this为JavaScript里的关键字，代表当前对象。

第8讲 JavaScript中的数组类型

   Java中的数组是固定长度，并且只能存放同一类型的数据。这些在JavaScript中都没有那么严格的限制，JavaScript里的数组长度不固定，可以任意添加数据，对数据类型也没有特殊限制（任意类型都可）。

1. 对象创建

    通常我们会采用JSON串的方式创建数组，数组创建时没有固定的长度，元素可以随时任意添加。具体语法如下：

    var 数组名 = [元素1，元素2，元素3，..... ]；

    还有另外一种创建方式，类似Java一样new Array()创建。

    var arr = new Array();

2. 数组访问

  1）通过下标访问（打印，赋值）里面的一个元素； ----> arr[2];

  2）通过for循环遍历，下标从0开始，最大下标arr.length-1

     for(int i=0;i<arr.length;i++){ .......   }

  3）遍历数组的所有元素，使用for..in语法

             for( 下标 in 数组){

                   alert( 数组名[下标] );

                }

     循环每执行一次，都会从数组中取得一个下标赋值给变量。

     通过上述访问，我们会发现，在JavaScript里对象和数组本质是一样的，都可以存放多个不同类型的数据。差别只是数组用下标访问，对象用属性名访问。

3. 数组的常见函数

    push():在数组结尾处添加一个元素

    pop() : 删除数组最后一个元素,并且返回这个元素值

    shift():删除数组第一个元素,并将元素值返回

    unshift():在数组开始位置(第一个元素之前)添加一个元素

    reverse():颠倒数组的数据

    join(str):将数组元素用str连接成字符串返回

    sort():对数组元素排序 , 默认按照字母顺序升序排列

      如果需要按照其他顺序排列,需要在调用sort()给定一个函数

      array.sort( function(v1,v2){ return v1-v2};  );

第8讲 JavaScript中的其他常见数据类型

1. Math类型常见函数

      abs()   绝对值

      sqrt()   开平方

      round()   四舍五入

      ceil()   向上靠拢截取数据

      floor()   向下靠拢截取数据

      random()  获得0--1的随机数

2. String类型常见函数

     str.indexOf("a");  在str里查找字符串a,如果a出现返回a的下标,如果没有

                         返回-1

      str.charAt(1);     获取下标为1的字符,str的初始下标0

      str.split();       拆分字符串

     str.substring(begin,length);   截取子串

     str.charCodeAt(2);  返回指定位置字符的ASCII编码

     str.replace('s','b');   将str里的第一个s字符替换成d

      注意：在javascript里求字符串的长度用length属性.

3. 日期类型

1）创建

    var  time = new Date();   -----> 用来获取当前系统时间

       var  time2 = new Date("1996-07-12"); ---> 创建一个指定时间的日期对象

2）常见函数

           getFullYear()   获得四位的年

            getDate()       获取日期里的日,返回值介于(1--31)

           getMonth()      获取日期里的月,返回值介于(0--11)

            getDay()        获取日期里的星期,返回值介于(0--6)

            getTime()       获得日期的毫秒表示

           toLocalString()  将日期转换为本地时间表示
