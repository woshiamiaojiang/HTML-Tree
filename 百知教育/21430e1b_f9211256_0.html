请求servlet的三种方式

知识点

地址栏请求

http://ip:port/web应用名称/url-pattern

超链接请求

<a href='/web应用名称/url-pattern'>test</a>

<a href="/myweb/first">显示当前时间</a>

Ø  获取client的数据：

通过超级链接的URL可以向servlet传递数据

<a href=“/myweb/first?name=tom”>传递数据</a>

在servlet中通过request对象获得数据

String name = request.getParameter(“name”);

由浏览器提交给服务器的数据称作客户端参数，习惯上也称作parameter，由一个名字和一个值组成，在servlet中依据名字获取参数的值，request.getParameter(“参数名”)。

表单请求

<form action='/web应用名称/url-pattern'>.....</form>

Ø  获取client的数据：

String str = request.getParameter("表单元素名/参数名");

Ø  解决post方式发送数据的中文乱码：

request.setCharacterEncoding(“GBK”);

常见问题

1、超链接后，传递的参数中的中文，显示时出现乱码

修改 tomcat根目录/conf/server.xml中的配置，增加URIEncoding属性：

<Connector port="8989" protocol="HTTP/1.1"

connectionTimeout="20000" redirectPort="8443" URIEncoding="GBK"/>

2、表单中输入的中文，显示时出现乱码

在获取 client 数据前，添加代码：

request.setCharacterEncoding("gbk");
