mvc

知识点

设计思想，为软件设计问题提供标准的解决方案。

MVC 将应用程序开发分成三个模块：

       Module(模型)：用于表示数据和业务逻辑；

       View(视图)：提供输入/输出界面；

       Controller(控制器)：控制应用程序流程，从哪来？干什么？去哪里？

                                                        // 1、获取 View 中的数据

                                                        // 2、调用 Module ，实现功能

                                                        // 3、根据结果，到 View 显示

                                                       

Model II 设计体系：遵循MVC设计思想，用JAVA语言进行的实现。

       Module 模型： 普通java类（JavaBean），对应 service和dao。

       View 视图： html/jsp

       Controller 控制器： servlet

示例，思路分析：

       login.jsp --> LoginAction.java  // 1、获取 client 数据

                                                 // 2、调用业务类的方法，实现“验证”功能

                                                 // 3、根据结果，跳转

                            --> LoginOk.jsp/error.jsp

       add.html --> AddAction.java  // 1、获取 client 数据

                                                 // 2、调用业务类的方法，实现“验证”功能

                                                 // 3、根据结果，跳转

                     --> QueryAllAction.java //[1、获取 client 数据]

                                                 // 2、调用业务类的方法，实现“验证”功能

                                                 // 3、根据结果，跳转

                     --> queryAllView.jsp(EL+JSTL)
