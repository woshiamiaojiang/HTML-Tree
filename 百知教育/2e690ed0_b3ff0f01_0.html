第1讲 JavaScript简介

    JavaScript最初由Netscape设计研发，最初被命名为LiveScript，后来Netscape在与Sun合作之后将其改名为JavaScript。JavaScript最初受Java启发而开始设计的，目的之一就是“看上去像Java”，因此语法上有类似之处，一些名称和命名规范也借自Java。之后，Ecma国际以JavaScript为基础制定了ECMAScript标准，完整的JavaScript实现包含三个部分：ECMAScript（描述了该语言的语法和基本对象），文档对象模型（DOM，描述处理网页内容的方法和接口)，浏览器对象模型（BOM，描述与浏览器进行交互的方法和接口）。

          

    JavaScript是一种运行在浏览器上得脚本语言。用于为静态网页(HTML)添加动态功能。如表单验证或简单用户交互。就像Java程序以Java虚拟及为运行环境一样，JavaScript是以浏览器为运行环境的。也就是说JavaScript代码中所涉及的计算，控制等功能都是由浏览器提供的。所谓脚本语言，就是说JavaScript源码不需要编译。他得源码会由浏览器直接执行。这样有好处，就是执行比较简单。当然也有不好的地方，如果代码中包含错误的话，就没有编译器的检查和提示了。程序员要依据浏览器的提示更正错误。这可能和以前的Java编程习惯有些区别。JavaScript 是互联网上最流行的脚本语言，这门语言可用于 HTML 和 web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

第2讲 JavaScript基本语法特征

1.      JavaScript的引入方式

    由于JavaScript是再浏览器中执行的。所以通常JavaScript代码会嵌入到html文件中：  

    方式一：代码直接嵌入

       <html>

   <head>

       <script type="text/javascript">

          //在这里嵌入JavaScript代码。

   </script>

   </head>

        </html>

    这种方式编写的JavaScript代码只能在一张HTML页面中使用。如果在另外一个页面上也要使用的话，需要将代码在另一个页面中重写一次。这样在代码维护上非常不利。所以，如果JavaScript代码需要在多个页面之间重用的话，我们可以通过另外一种方式。

    方式二：将代码写在一个独立的文件中，然后在需要使用JavaScript代码的页面中引入这个文件。如下所示：

<html>

   <head>

     <script type="text/javascript" src="js/functions.js"></script>

         </head>

       </html>

    上例中，JavaScript代码被写在一个叫functions.js的文件中。这样文件中的代码就可以在这个页面中使用了。

5.      JavaScript变量的使用

    Javascript变量也需要先声明后使用，同时，如果一个变量的值在使用之前没有进行初始化，将会被当做”undefined”处理。”undefined”是javaScript中一个特殊的字面值，表示这个值未做定义。

   1) 变量允许重复声明定义and赋值, 值以最后一次为准

       var  v1=10;

       var  v1="hello";

   2) JavaScript里没有块作用域

      java:

          if(true){   int a=10; }

          System.out.println(a);    ---> 编译错误

     javascript:

          if(true){  var  a=10; }

          alert(a);

   3) 局部变量: 只有定义在函数里 and 用var声明的变量

   4)不用var声明定义的变量（隐式声明）,直接是全局变量

第6讲 JavaScript中的函数

    和其他语言类似，函数是非常重要的功能模块，在JavaScript里函数被认为是一种数据类型，一个函数可以被看做是一个值（一个函数类型值）。所以JavaScript中函数定义比较灵活，可以通以下几个方式来实现：

  1. 通过关键function声明定义函数

     function add( a , b ){

                return a + b ;

      }

   “function”是声明一个函数的关键字，add是函数的名字。括号内的a , b 是两个参数。需要注意的是参数不需要指定数据类型，因为JavaScript中所有参数都是以var声明的。另外就是再函数声明中不用指定返回类型。函数的调用形式和Java一样。如 ：

     var r = add( 10 , 20 ) ;

     alert(r )

我们看到的结果是30 ;

  2. 函数值之间直接赋值

    在JavaScript中，函数也可以通过变量的形式来调用。这和C/C++中的函数指针类似：

  function add( a , b ){

            return a + b ;

  }

 /* 将add函数看做一个普通的值，是值就可以在变量之间互相赋值。赋值成功后

    v1变量里也保存了一个函数类型值，那么v1也可以当做函数来使用 */

  var otherName = add ;

  var r = otherName( 10 , 20 ) ;

  alert(r )

我们看到的值依然是30 。

注意： var v1 = add;  //将add变量里保存的函数值付给v1变量

      var v2 = add();  // 将add函数的执行结果赋值给v2变量

 3. 匿名定义函数

    var a = function ( x , y ){ return x + y; } ;

    在上面例子中，可以理解为a就是这个函数的名字了，对进行调用就相当于调用定义的函数。如 var r = a( 10 , 20 ) /；这种形式也是以后用的比较多得一种形式。

 4. 构造函数
