JavaScript 模块三 DOM编程

第1讲 DOM的基本概念

    在前面的例子中我们看到，通过JavaScript虽然可以响应页面中的一些事件，但是几乎不能做任何有意义的操作，主要的原因就是我们无法通过JavaScript对页面上的元素进行操作。这大大限制了JavaScript的能力。下面我们要学习的DOM编程，就是让JavaScript通过Dom对象操作页面上的元素。

    首先我们先来了解一下什么是DOM对象。DOM（documentobjectmodel）称为“文档对象模型”。它将一个html文档看成是由若干个元素组成，网页中的每一个标签元素都被理解为一个标签对象。如下所示：

    <html>

          <body>

                 <img id=”sample” src=”./a.jpg” width=”100” height=”100”/>

              </body>

    </html>

对于上述文档，其中的主要元素就是一个 img 标签。在JavaScript代码里，可以把img标签对应成一个JavaScript的对象。标签中得每个属性就是这个对象的属性，我们可以通过修改对象的属性，来改变标签在页面上的特征。例如将对象的width值变大，这样可以使图片在页面上的显示尺寸变大；修改src的值，让页面显示另外一幅图片。

其次，页面上的元素实际上是彼此关联的。比如 ：

 <select id="a" name="a" onchange="doChange(this)">

       <option value="1">11111</option>

       <option value="2">22222</option>

       <option value="3">33333</option>

</select>

我们可以看出，这段html代码中一共有4个元素组成，一个select 元素 ，和三个option元素，而且三个option元素是属于select 元素的。这时我们把select叫做父亲元素，三个option元素 称作 select元素的 子元素。DOM不仅可以代表单个的页面元素，还可以表示元素和元素之间的关系。实际上DOM元素是将整个HTML文档描述成一个树状结构。<html>标签就是这个树状结构的根。比如下列html代码：

   <html>

<head></head>

<body>

              <select id="a" name="a" onchange="doChange(this)">

                     <option value="1">11111</option>

                     <option value="2">22222</option>

                     <option value="3">33333</option>

              </select>

              <select id="b" name="b">

                     <option value="a">aaaa</option>

                     <option value="b">bbbb</option>

                     <option value=”c”>cccc</option>

              </select>

              <div id="c">hello world</div>

</body>

</html>

第2讲 DOM常见函数

1. 用来查找DOM对象的函数

   1）var tag = document.getElementById("标签id属性的值");         

      Element 根据id查找一个DOM对象

   

   2）var tags = document.getElementsByTagName("标签名");

     该方法返回值为array , 根据标签名查找DOM对象，由于标签名可以重复，所以返回一个数组。例如：

    eg：<h1> hehehehehehe </h1>

               <p id="p1">

                      <h1>hahahahahah</h1>

                   <h1> heiheiheihei </h1>

            </p>

               var tags = document.getElementsByTagName("h1"); // tags.length=3;

        var tag = document.getElementById("p1");  // 一个标签对象

            var tags2 = tag.getElementsByTagName("h1"); // tags2.length=2

   3）var tag = document.getElementsByName("标签的name属性值");

     该方法根据标签的name属性获取对应的标签对象。

注意：获得标签后

    a. 可以修改标签的属性

       b. 可以修改标签的显示风格  tag.className="css类别名称"

2. DOM对象的常见属性

1）this --> 代表当前产生事件的标签对象

2）innerHTML ---> 代表标签体里的内容（子标签 or 文本内容）

3）element.childNodes       获得element的所有子节点

4）element.firstChild         获得element的第一个子节点

5）element.lastChild          获得element的最后一个子节点

6）element.nextSibling       获得element的下一个（右侧）兄弟节点

7）element.nodeValue        获得字符节点内容(element必须是字符节点)

8）element.parentNode       获得element的父节点

9）element.previousSibling    获得element的前一个(左侧)兄弟节点。

2. DOM中用来修改文档结构的方法

1）删除标签

             parentTag.removeChild(被删除的子标签对象);

   删除辅助功能：确认框

                var v1 = window.confirm("提示信息");  ---> v1是一个boolean值

2）替换标签

             parentTag.replaceChild(新标签，旧标签);

3）创建新标签

             document.createElement("标签名");

4）创建文本内容节点

             document.createTextNode("文本内容");

5）增加一个子节点，新的节点位于所有子节点的最后(最右侧)

             parentTag.appendChild(子标签或者文本节点);

              eg:  <h1> hehehehe </h1>

                  ---> var tag = document.createElement("h1");

                          var txt = document.createTextNode("hehehehe");

                             tag.appendChild(txt);

                             

6）将一个新的子标签插入到某一个子标签之前

             element.insertBefore( newTag,oldTag);

在element的某个子节点之前插入一个新的子节点, newTag为新插入的子节点，oldTag是element的一个原有子节点，新节点会插入在它前面。

7）element.hasChildnodes().判断element是否有子节点。
