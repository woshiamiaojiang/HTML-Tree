第1讲 Struts 2 接收请求参数

课程内容

Struts使用属性以及属性的set方法来接收各种类型的请求参数， Struts 支持转换的类型和属性以及请求参数的对应关系见下表：

 

                                                                       
 

类型

  		 

属性名（set方法名称）

  		 

请求参数格式

 
 

String

  		 

String name

  		 

name=zhangsan

 
 

Boolean/boolean

  		 

Boolean sex

  		 

sex=true

 
 

Character/char

  		 

Character score

  		 

score=A

 
 

Integer/int,Long/long, Float/float,   Double/double

  		 

Double salary

  		 

salary=4000.0

 
 

BigDecimal/BigInteger

  		 

java.math.BigDecimal salary

  		 

salary=4000.0

 
 

Enum

  		 

Gender gender

 

（例如有枚举类Gender,枚举值MALE、FEMALE）

  		 

gender=MALE

 
 

dates

  		 

java.util.Date birthday

  		 

birthday=14-6-16 上午11时10分03秒

 

birthday=14-6-16 11:10:03

 

birthday=14-6-16 上午11:10

 

birthday=14-6-16

 

birthday=2014-6-16（推荐）

 

birthday=2014年6月16日

 

birthday=2014-06-16T11:10:03（推荐）

 

对于日期类型转换，struts支持7种格式，可以参考DateConverter这一实现

 
 

Arrrays,List

  		 

String[] list 或

 

java.util.List<String> list

  		 

list=值1&list=值2...&list=值n 或

 

list[0]=值1&list[1]=值2...&list[n]=值n

 
 

Map

  		 

Map<Integer, String> map

  		 

map[0]=北京&map[1]=天津...&map[n]=上海

 
 

实体类

  		 

Student student

 

（例如student类中有属性id,name）

  		 

student.id=20&student.name=张三

 
 

List组合实体类

  		 

List<Student> list

  		 

list[0].id=20&list[0].name=张三&
  list[1].id=21&list[1].name=李四

 

课程案例

对于ParameterAction类

 
 

public class ParameterAction {

 

       private User user;

 

}

 

和User类

 
 

public class User {    

 

       private String name; 

 

       private int age;  

 

       private Date birthday;    

 

       private boolean sex; // true 男 false 女

 

    private List<String> hobby;

 

}

 

 

要正确给Action中的user属性赋值，请求参数格式应当是怎样的，如果使用表单传参，表单该如何编写？

解答：

a)        get请求方式

 
 

user.name=zhang&user.age=18&user.birthday=2001-1-1&user.sex=true

 

 

b)       表单请求方式

 
 

<form action="/struts2/day2/parameter"  method="get">

 

              <input type="text" name="user.name"/>

 

              <input type="text" name="user.age"/>

 

              <input type="text" name="user.birthday"/>

 

              男 <input type="radio"  name="user.sex"  value="true"/>

 

              女 <input type="radio"  name="user.sex"  value="false"/>

 

             

 

              抽烟<input type="checkbox"  name="user.hobby"  value="抽烟"/>

 

              喝酒<input type="checkbox"  name="user.hobby"  value="喝酒"/>

 

              看球<input type="checkbox"  name="user.hobby"  value="看球"/>

 

              <input type="submit" value="提交">

 

</form>

 

 

提高：如果Action中想在一次请求接收多个用户对象，则按在Action中添加属性List<User> list和get,set方法，并按如下格式传递请求参数

 
 

list[0].name=zhang&list[0].age=22&list[0].birthday=1999-1-1&list[0].sex=false&

 

list[1].name=lisi&list[1].age=17&list[1].birthday=1998-1-1&list[1].sex=true

 

 

0下标代表第一个user, 1下标代表第二个user ....
