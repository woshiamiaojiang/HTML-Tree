监听器

知识点

Listener 称作监听器，当web应用中某一特定事件发生时会调用相应的Listener，常见的监听器分为2类：

Ø  生命周期对象监听器：监听 生命周期对象 的创建和销毁

Ø  命名属性监听器：监听 生命周期对象中命名属性的添加、修改和删除

自定义监听器的步骤：

1、写代码

生命周期对象监听器：implements ServletRequestListener

implements HttpSessionListener

implements ServletContextListener

命名属性监听器：implements ServletRequestAttributeListener

implements HttpSessionAttributeListener

implements ServletContextAttributeListener

举例：

public class MySessionListener implements HttpSessionListener {

    // 当HttpSession 创建时调用

    public void sessionCreated(HttpSessionEvent arg0) {

       System.out.println("=======sessionCreated=========");

    }

    // 当HttpSession 销毁时调用

    public void sessionDestroyed(HttpSessionEvent arg0) {

       System.out.println("----------- sessionDestroyed -----------");

    }

}

2、配置 web.xml

    <!-- 监听器 -->

    <listener>

       <listener-class>com.zpark.lns.MySessionListener</listener-class>

    </listener>
