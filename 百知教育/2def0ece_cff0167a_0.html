ServletContext和 ServletConfig 接口

1）ServletContext 接口

              是web.xml文件在内存中的表现。

              和web应用 一一对应。

                    

              应用：获取全局初始化参数

              a、配置全局初始化参数 web.xml

                     在所有<serlvet>标签外面。

                     <context-param>

                            <param-name>name</param-name>

                            <param-value>zs</param-value>

                     </context-param>

              b、读取全局初始化参数

                     // 获取 SerlvetContext 对象

                     ServletContext ctx = super.getServletContext();

                     // 读取全局初始化参数

                     String name = ctx.getInitParameter("name");

                    

2）ServletConfig 接口

              代表，web.xml中的一对<servlet></serlvet>标签。

              和 servlet 一一对应。

             

              应用：获取局部初始化参数

              a、配置局部初始化参数 web.xml

                     在<servlet>标签内部。

                     <servlet>

                            <servlet-name>LocalParamServlet</servlet-name>

                            <servlet-class>com.zpark.yyl.servlet.LocalParamServlet</servlet-class>

                            <init-param>

                                   <param-name>age</param-name>

                                   <param-value>10</param-value>

                            </init-param>

                     </servlet>

              b、读取局部初始化参数

                     // 获取 SerlvetConfig 对象

                     ServletConfig conf = super.getServletConfig();

                     // 读取局部初始化参数

                     String strAge = conf.getInitParameter("age");
