3、获取命名属性【重点】

       1) 获取指定作用域中的命名属性

              ${requestScope.myStr }

              等价于jsp中的代码：obj = request.getAttribute("myStr");

                                                 String str = obj.toString();

              EL中的作用域对象：pageScope、    requestScope、sessionScope、applicationScope

       2)从作用域中搜索命名属性

              ${myStr}

              等价于jsp中的代码：obj = pageContext.findAttribute("myStr");

                                                 String str = obj.toString();

       3)获取命名属性中的对象属性（命名属性中存储的是实体类对象）

              ${requestScope.person.name }

              等价于jsp中的代码：obj = reqeust.getAttribute("person");

                                          Person p = (Person)obj;

                                          p.getName()

4、获取cookie的值

       ${cookie.名称} 根据名称获取对应的cookie对象。如,${cookie.JSESSIONID}

       ${cookie.名称.value} 根据名称获取对应cookie对象的值。如,${cookie.JSESSIONID.value}

       等价于jsp中的代码：

              Cookie[] cookies = request.getCookies();

              if (cookies != null){

                     for(int i=0; i<cookies.length; i++){

                            Cookie c = cookies[i];

                            //获取名字是JSESSIONID的cookie对象

                            if (c.getName().equals("JSESSIONID")){

                                   //获取名字是JSESSIONID的cookie对象的值

                                   out.println(c.getValue());

                            }

                     }

              }

5、pageContext对象

       注意：jsp的隐含对象只能在<%= %>和<%%>中使用，不能在EL中使用。

       该对象在JSP和EL中都有，但含义不同：

       1）JSP中的pageContext，代表页面作用域对象；

       2）EL中的pageContext，提供了在EL中可以使用JSP隐含对象的一种方式。

              如，${pageContext.session.id} 等价于 <%=session.getId() %>

              ${pageContext.request.contextPath}等价于 <%=request.getContextPath() %>

6、访问数组或集合元素(命名属性中的数组和集合)

       ${myArr[0]}         获取命名属性myArr中数组的第1个元素

       ${myList[0]}         获取命名属性myList中list集合的第1个元素

       ${myMap["键的名称"]}       根据键的名称获取命名属性myMap中的对应键值

       ${myMap.键的名称}

示例：

<%

        //测试数据

        // list集合

        List<User> list = new ArrayList<User>();

        list.add(new User("zs","111"));

        list.add(new User("ls","222"));

        list.add(new User("ww","333"));

        pageContext.setAttribute("myList",list);

        // map集合

        Map map = new HashMap();

        map.put("aaa",new User("zs","111"));

        map.put("bbb",new User("ls","111"));

        map.put("ccc",new User("ww","111"));

        pageContext.setAttribute("myMap",map);

    %>

<!-- 使用EL操作命名属性中的数组和集合 -->

${myList[1].username }<br/>

${myMap["ccc"].username }<br/>

${myMap.aaa.username }<br/>
