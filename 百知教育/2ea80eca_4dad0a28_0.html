CSS

知识点

语法

样式规则的语法：

       样式属性名:值;样式属性名2:值2;

行内样式

场景：应用于网页中的一个元素样式特殊。

语法：<html标签 ... style="样式规则" >...</html标签>

应用：自动应用

内嵌样式

<head>

       <style type="text/css">

              选择器名称 {

                     样式规则

              }

       </style>

</head>

1、html选择器

       场景：同一个网页中的多个相同标签，样式一致。

       语法：<style type="text/css">

                            html标签名称 {

                                   样式规则

                            }

                     </style>

       应用：自动应用

2、class选择器(类选择器)

       场景：同一个网页中的多个不同标签，样式一致。

       语法：<style type="text/css">

                     .选择器名称 {

                            样式规则

                     }

              </style>

       应用：<html标签 ... class="选择器名称(不含.)"> ... </html标签>

             

3、id选择器（了解）

       场景：同一个网页中的多个不同标签，样式一致。

       语法：<style type="text/css">

                     #选择器名称 {

                            样式规则

                     }

              </style>

       应用：<html标签 ... id="选择器名称(不含#)"> ... </html标签>

       注意：在html标签中，id属性，通常用于唯一标识网页中的一个页面元素。

             

4、伪类选择器

       场景：在特定的标签上，产生特定动作时，样式特殊。

       语法：

                     a:hover{    /* 鼠标移上 */

                            样式规则

                     }

       应用：自动应用

      

       常见的伪类选择器名称：

              a:hover           /* 鼠标移上 */

              a:visited         /* 访问过的超链接 */

              a:active          /* 正在点击的超链接 */

              a:link                    /* 未访问过的超链接 */

外部样式

场景：同一站点内的多个网页，样式风格统一。

语法： 

       1）定义外部的样式表文件

              后缀 .css

              文件中不能包含html标签，只能有css相关语法

       2）引入外部表文件

              <head>

                     <link rel="stylesheet" type="text/css" href="css/mystyle.css" />

              </head>

应用：同 “上面各种样式表”
