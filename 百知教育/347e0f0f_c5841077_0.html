jsp的隐含对象

知识点

由tomcat 创建，可以在JSP中的 <%=%> 和 <% %> 中使用的一组对象。

1、request

       类型：HttpServletRequest

       方法： str = request.getParameter("name");//获取client数据

                            request.setAttribute("str",obj);

String str = request.getContextPath();//获取web应用名称

......

示例：

<%@ page pageEncoding="UTF-8"contentType="text/html"%>

getProtocol=<%=request.getProtocol() %><br/>

getMethod=<%=request.getMethod() %><br/>

getRequestURL=<%=request.getRequestURL() %><br/>

getRequestURI=<%=request.getRequestURI() %><br/>

getContextPath=<%=request.getContextPath() %><br/>

运行结果：

2、response

       类型：HttpServletResponse

       方法：response.setContentType("text/html");

                     response.setCharacterEncoding("UTF-8");

                     PrintWriter out = response.getWriter();

3、out

       类型：JspWriter

       方法： out.print("");

                     out.println("");

                     out.write("");

4、session

       类型：HttpSession

       方法：session.setAttribute("",obj)

                     session.getAttribute("")

                     session.removeAttribute("")

5、application

       类型：ServletContext

       方法：application.getInitParameter("");//获取web.xml中配置的全局初始化参数

                     application.setAttribute("",obj)

                     application.getAttribute("")

                     application.removeAttribute("")

6、config

       类型：ServletConfig

7、page

       类型：Object

       代表当前JSP页面本身，相当于 this。

8、exception

       类型：Throwable

       注意：该对象只能要 isErrorPage="true" 的页面中使用。

示例： error.jsp

<%@ page pageEncoding="UTF-8"contentType="text/html"%>

<%@ page isErrorPage="true" %>

出错啦！！！<br/>

<%=exception.getMessage() %><br/>

jsp1.jsp

<%@ page pageEncoding="UTF-8" contentType="text/html" %>

<%@ page errorPage="/error.jsp" %>

<%    

           int result = 5/0;

    %>

运行结果：

9、pageContext

       类型：javax.servlet.jsp.PageContext

       1）获取 其它 8个 隐含对象

              pageContext.getRequest();            // 获取到 request 对象

              pageContext.getServletContext();  --> application

       2）作用域对象

              开始：请求到达该页面开始

              结束：请求离开该页面

使用：pageContext.setAttribute(“str”,obj);

obj = pageContext.getAttribute(“str”);

示例：

forward1.jsp

<%@ page pageEncoding="UTF-8"contentType="text/html"%>

这是forward1页面<br/>

<%

               request.setAttribute("name","zs");

               pageContext.setAttribute("name","tom");

       %>

<jsp:forward page="/forward2.jsp"/>

forward2.jsp

<%@ page pageEncoding="UTF-8"contentType="text/html"%>

这是forward2页面<br/>

<%=request.getAttribute("name") %><br/>

<%=pageContext.getAttribute("name") %><br/>

 

运行结果：

       3）操作其它 3 个作用域对象

              // 使用pageContext往request作用域中设置命名属性

              pageContext.setAttribute("str",obj,PageContext.REQUEST_SCOPE);

              // 使用pageContext获取session作用域中的命名属性

              obj = pageContext.getAttribute("str",PageContext.SESSION_SCOPE);

              // 使用pageContext删除application中的命名属性

              pageContext.remove("str",PageContext.APPLICATION_SCOPE);

             

              // 在jsp的作用域中从小到大进行搜索，搜索即停止

              obj = pageContext.findAttribute("str");

查所有功能

需求描述：查询数据库中的所有帐户信息，并以表格的形式展现在界面中；如果没有查询到数据，则显示“暂无数据！”。

分析：    查询所有帐户 - queryAllAction.jsp

显示所有帐户信息 - queryAllView.jsp

关键代码：

（queryAllAction.jsp）

<%@page import="com.zpark.entity.Account"%>

<%@page import="java.util.List"%>

<%@page import="com.zpark.service.BankServiceImpl"%>

<%

    // [1获取client数据]

    // 2调用业务类，实现“查所有”功能

    BankServiceImpl bankService = new BankServiceImpl();

    List<Account> accList = bankService.queryAll();

    // 3根据结果，跳转

    request.setAttribute("myList", accList);

%>

<jsp:forward page="/queryAllView.jsp"/>
