第2讲 SELECT语句【重点】

基础语法：select 字段名1,字段名2,... from 表名；

1. 查询部分列

   eg： 查询员工的工号、姓、工资、入职时间

              select employee_id,last_name,salary,hire_date from employees;

2. 查询所有列

   eg：查询员工的所有信息

         方法一：select employee_id,first_name,last_name,email,phone_number,hire_date,

                       job_id,salary,commission_pct,manager_id,department_id from employees;

         方法二：select * from employees;

       注意： *在开发时不建议使用，只在学习阶段练习使用。

          原因：语义不明确；会把不需要的列也查出来，影响性能。

3. 对查询出的列进行数学运算（加减乘除）

   eg：查询员工的工号、姓、工资、所在部门、年薪(月薪*12)

          select employee_id,last_name,salary,department_id,salary*12 from employees;

   注意：

u  在sql命令里，字符串类型值不能做加法操作。如果需要做字符串连接，会有其他符号完成。

u  对于日期类型值，可以做加减法操作。运算单位以天为单位。

4. 可以为查询的列起别名

   语法： select 字段名 as 别名，表达式 [as] 别名 from 表名；其中as可以        

         省略。

    eg：上题可以改写为

        select employee_id "id",last_name as name,salary,department_id,salary*12 as "年薪"    

        from employees;

     注意：别名中如果出现空格或使用sql中的关键字/保留字，则需要使用“”引起来。

5. 字符串连接 ||

    在employees表中，一个人的名字由first_name , last_name 两列组成，如果我们想把两列数据合并成一列数据的话，就可以使用字符串连接函数了。

select first_name || '-' || last_name as name , employee_id as id , email from employees

   

   注意：oralce中的字符串常量以及日期常量，使用‘ ’(单引号) 引起来。

6. 去除查询结果里的重复数据

  语法：将distinct放在select后面，可以去除结果里的重复数据

   eg:  查询公司里所有的姓（姓唯一）

      select  distinct  last_name from  employees;

      查询公司里所有的姓名。（姓和名联合唯一）

      select  distinct  last_name,first_name  from employees;

 

第3讲 ORDER BY语句【重点】

语法：select ... from ... order by ... ； Order by 语句用于对查询结果进行排序

1. 单列排序。

1)       指定时可以是列的列名、列的别名、或者是列在select里出现时的下标。

      eg：按工资从大到小显示员工的工号、姓、收入

          select employee_id,last_name,salary from employees order by salary; --> 指定列名

      select employee_id,last_name,salary from employees order by 3; --> 列出现时的下标

2)       降序排列时使用关键字desc（将descend放在需要降序排列的列名后面）

      eg：按工资从大到小显示员工的工号、姓、收入

       select employee_id,last_name,salary from employees order by salary desc;

      注意：null也参与排序，在oracle里是最大值。

2. 多列排序

   语法：Order by 还可以根据多个列进行排序，参与排序的类名字依次写在order by 的后 

        面，列名与列名之间用”,”隔开。排序时先按照第一列进行排序，第一列取值相同

        的再按照第二列取值排序，依次类推

Ø  select employee_id,last_name,salary from employees order by first_name,last_name;

       (先按first_name升序排列，如果一样的first_name,在按last_name升序排列)

Ø  select * from employees  order by first_name,last_name desc;

        (先按first_name升序排列，如果一样的first_name,在按last_name降序排列)

Ø  select * from employees  order by first_name desc,last_name desc;

       (先按first_name降序排列，如果一样的first_name,在按last_name降序排列)
