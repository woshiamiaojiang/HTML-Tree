第2讲 值栈（ValueStack）

课程内容

1.  值栈的组成

值栈的通俗概念：是Struts2自己设计的一种作用域，它的作用范围仅限于一次请求。

值栈由两部分组成：ROOT（根）部分，Context(上下文) 部分

2.  获取值栈

ValueStack vs = ActionContext.getContext().getValueStack();

3.  操作值栈的Root部分

值栈的Root部分的数据结构是“栈”，类似于List，符合“后进先出”的规则

vs.push(对象);// 向Root压入对象至栈顶

（这里最好画一个示意图，先压入的值在栈的底部，后压入的值在栈的顶部）

Object对象= vs.pop(); // 从Root栈顶弹出对象

Object对象= vs.peek(); // 获得Root栈顶对象，不弹出

 

实例1：存字符串值

           
 

java代码

  		 

public String execute() {

 

       ValueStack   vs = ActionContext.getContext().getValueStack();

 

       vs.push("字符串对象");

 

       return "success";

 

}

 
 

jsp代码

  		 

<s:property  />

 
 

输出结果

  		 

字符串对象

 

实例2：存一个User实体对象

           
 

java代码

  		 

public String execute() {

 

       User   u1 = new User();

       u1.setAge(21);

 

       ValueStack   vs = ActionContext.getContext().getValueStack();

 

       vs.push(u1);

 

       return "success";

 

}

 
 

jsp代码

  		 

<s:property  value="name"/>

 
 

输出结果

  		 

王五

 

实例3：存多个User实体对象

           
 

java代码

  		 

public String execute() {

 

       User   u1 = new User();

 

       u1.setName("王五");

       u2.setName("张三");

jsp代码

  		 

<s:property  value="name"/>

 
 

输出结果

根据上三个实例，总结规律：<s:property />标签如果没给value属性，那么从表示从栈顶取出对象显示，如果给了value属性，按从上至下找栈中对象的属性，找到为止并显示。

在页面添加<s:debug />标签，观察debug标签所展示的Root栈结构，发现struts会将当前Action对象压入栈，由此解释的<s:property />取值原理，示例见下：

4.  操作值栈的Context部分

值栈的Context部分的数据结构是“Map”，以“键值对”的方式进行存取。Context中已经预置了一些常用对象的key，例如：request，session，application，action等，当然也可以自己向Context中存入新的key和value，要注意OGNL如果从Context中取值，语法为“#key”，以区别从Root部分按属性名取值

实例1：存取session中的值

           
 

java代码

  		 

public String execute() {

 

       Map<String,   Object> map = ActionContext.getContext().getSession();

 

       map.put("name", "zhangsan");

 

       return "success";

 

}

 
 

jsp代码

  		 

<s:property  value="#session.name"/>

 
 

输出结果

  		 

张三

 

实例2：存取application中的值

           
 

java代码

  		 

public String execute() {

       return "success";

 

}

 
 

jsp代码

实例6：使用OGNL配合ValueStack存值，主要是利用了ValueStack.setValue方法，它可以作为一种通用的存值方法，既可以向Context存值，也可以向Root存值：

           
 

java代码

  		 

public class ContextAction {

 

       private String password;

 

       public String execute() {

 

         ValueStack vs = ActionContext.getContext().getValueStack();

 

              // 参数1: 是一个ognl表达式, 参数2: 要存储的值

 

              vs.setValue("#request.ccc", "ddd");

 

              vs.setValue("#session.eee", "fff");

 

              vs.setValue("password", "123456");

 

              return "success";

 

       }

 

}

 
 

jsp代码

  		 

<s:property  value="#request.ccc"/><br/>

 

<s:property  value="#session.eee"/><br/>

 

<s:property  value="password"/><br/>

 
 

输出结果

  		 

ddd

 

fff

 

123456
